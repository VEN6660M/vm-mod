# VMMOD — Official Website

Landing page для [VMMOD Mod Manager](https://github.com/VEN6660M/venom).

## Файлы

```
index.html   — главная страница
style.css    — стили
script.js    — скрипты (анимации, меню)
```

## Деплой на GitHub Pages

1. Загрузи все три файла в репозиторий (можно в корень или в папку `docs/`)
2. Зайди в **Settings → Pages**
3. В разделе **Source** выбери:
   - Branch: `main` (или `master`)
   - Folder: `/ (root)` — если файлы в корне, или `/docs` — если в папке docs
4. Нажми **Save**
5. Через 1–2 минуты сайт будет доступен по адресу:
   `https://<username>.github.io/<repo>/`
